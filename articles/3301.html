AWS 向 AI Agent 開放其雲端服務

來源：Heise Online · DE/de · 閱讀原文 ↗

摘要

Amazon Web Services（AWS）正式推出一款供 AI Agent 使用的 MCP（Model Context Protocol）伺服器，並已開放給所有用戶使用。此舉旨在解決 AI Agent 在操作 AWS 時常遇到的核心問題：這些 Agent 的訓練資料往往已過時，導致無法正確理解或使用 AWS 最新的服務與 API。透過 MCP 伺服器，AI Agent 可以即時取得最新的 AWS 文件與操作資訊，從而更準確地執行雲端任務。這對企業在自動化雲端管理、基礎建設部署等方面具有重要意義，有望大幅提升 AI Agent 與 AWS 服務整合的可靠性。
