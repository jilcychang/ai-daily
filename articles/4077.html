Anthropic 為 Claude Code 新增多 Agent 管理功能「Agent View」

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

Anthropic 於 5 月 11 日以研究預覽（research preview）形式，為其 AI 程式開發工具 Claude Code 推出全新功能「Agent View」。這項功能讓開發者能在單一介面中同時管理多個 Claude Code 的 CLI（命令列介面）工作階段，大幅簡化平行運行多個 AI 編碼任務的操作流程。業界分析認為，此舉代表 Anthropic 正積極將 AI 開發環境提升至「Agent 作業系統」的層次，讓 Claude Code 不再只是單一對話工具，而是能夠統籌調度多個 AI Agent 協同作業的管理平台。這項發展也反映出 AI 輔助開發正朝向更高度自動化與多工並行的方向演進。
