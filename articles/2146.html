Google 修補 Gemini CLI 重大漏洞，未更新恐導致開發整合流程遭受遠端程式碼攻擊

來源：iThome 新聞 · TW/zh-TW · 閱讀原文 ↗

摘要

Google 近日透過 GitHub 發布資安公告，針對 Gemini CLI 搭配 GitHub Actions 使用時存在的重大安全漏洞發布修補更新。該漏洞若未修補或緩解，攻擊者可能藉由程式開發部署流程（CI/CD pipeline）發動遠端程式碼執行（RCE）攻擊，對開發團隊構成嚴重威脅。Google 強烈建議所有使用 Gemini CLI 的用戶立即更新至最新版本以降低風險。
