摘要

Cohere 共同創辦人 Nick Frosst 在 Reddit r/LocalLLaMA 親自發文，宣布推出 Command A+，回應社群數個月前詢問 Command-A 系列進展的疑問。Command A+ 是 Cohere 首款 MoE（混合專家）架構模型，主打高效率與速度，在同級模型中被定位為最快、回應最靈敏的選項之一。團隊針對量化技術做了大量優化，使該模型即便在僅有 1 至 2 張 GPU 的環境下也能流暢運行，大幅降低中小型團隊與開發者的使用門檻。模型採用 Apache 2.0 授權，幾乎不設任何使用限制，方便開發者自由構建 AI agent 應用。Nick Frosst 強調，Cohere 雖以企業客戶為主，但開源社群的回饋對其產品與模型方向影響深遠，也承認上次被社群「點名」後確實有所調整。此次發文邀請社群分享使用心得與專案，展現出企業與開源生態積極互動的姿態。
