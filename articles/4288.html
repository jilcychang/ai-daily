Anthropic 調整 Claude 訂閱用量計算，Agent SDK 等程式化使用改走專屬月額度

來源：iThome 新聞 · TW/zh-TW · 閱讀原文 ↗

摘要

Anthropic 宣布自 6 月 15 日起，將調整 Claude 付費訂閱方案中程式化使用的用量計算方式。Pro、Max、Team 與 Enterprise 方案用戶，可申領每月固定額度，專門用於 Claude Agent SDK、指令列非互動模式（claude -p）、Claude Code GitHub Actions，以及透過 Claude Agent SDK 搭配 Claude 訂閱方案使用的第三方工具。此變更意味著程式化／自動化使用情境將從一般對話用量中獨立出來，改以專屬月額度計算，有助於開發者更清楚掌控 API 與 Agent 工作流程的用量，避免與日常互動使用互相排擠。
