Anthropic 繼 OpenAI 之後，為 Claude 新增「自主目標」功能

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

Anthropic 在其開發者編程工具 Claude Code 中新增了 `/goal` 指令，進一步強化自主型 AI Agent 的能力。這項功能讓使用者只需設定完成條件，AI 便會跨越多個工作回合（turn），持續執行任務直到達成目標為止。根據 Anthropic 官方文件，`/goal` 是以工作階段（session）為單位設定特定完成條件的功能。使用者可以輸入如「所有測試通過」、「完成所有 API 呼叫遷移」、「清空 issue backlog」等明確目標，Claude Code 便會自主規劃並執行，直到條件滿足。此功能與 OpenAI 近期推出的類似自主任務機制相呼應，顯示頭部 AI 廠商正積極競逐更高層次的 Agent 自動化能力，讓開發者能以更低的介入成本完成複雜的工程任務。
