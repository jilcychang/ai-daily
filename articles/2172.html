OpenAI 放寬與 Microsoft 的獨家合約，模型可在其他雲端平台上架，Amazon Bedrock 率先解禁

來源：ITmedia AI+ · JP/ja · 閱讀原文 ↗

摘要

OpenAI 與 Microsoft 修訂了雙方的合作協議，原本限制 OpenAI 產品只能透過 Azure 提供的獨家條款已正式鬆綁。根據新協議，OpenAI 的全系列產品與模型將可在 Azure 以外的其他雲端平台上部署與提供服務。Amazon Bedrock 成為首個受惠的平台，即刻起可上架 OpenAI 的模型。此次協議調整對 AI 產業格局具有重要意義：對 OpenAI 而言，能觸及更廣泛的企業客戶群，尤其是已深度使用 AWS 生態系的企業；對 Microsoft 而言，雖然失去獨家優勢，但據報導此次協議重組與 OpenAI 的公益轉營利公司結構調整密切相關，Microsoft 也獲得了相應的利益補償。這標誌著 OpenAI 正積極拓展其商業版圖，不再依賴單一雲端通路。
