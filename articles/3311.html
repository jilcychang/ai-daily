AWS WorkSpaces：AI Agent 操控舊有應用程式

來源：Heise Online · DE/de · 閱讀原文 ↗

摘要

Amazon Web Services（AWS）宣布讓 AI Agent 能夠直接操作 WorkSpaces 虛擬桌面環境，透過模擬滑鼠點擊、鍵盤輸入以及螢幕畫面分析等方式與應用程式互動，完全不需要在目標應用程式中內建任何 API 接口。這項功能對於企業內部仍在使用的舊有（legacy）系統特別重要，因為這類系統往往缺乏現代化的 API 介面，過去難以進行自動化整合。透過 AWS 的這項新能力，AI Agent 可以像人類使用者一樣操作任何桌面應用程式，大幅降低企業導入 AI 自動化的技術門檻，讓老舊系統也能受惠於智慧化流程。
