Warpdrv - 我的開源 Llama.cpp 啟動器，在 Strix Halo + RTX Pro 上日常跑 Qwen 35b + 27b

來源：Reddit r/LocalLLaMA · US/en · 閱讀原文 ↗

摘要

Reddit 用戶 xornullvoid 開源了一款名為 Warpdrv 的 Llama.cpp 圖形化啟動器，專為在本地硬體上同時運行多個大型語言模型而設計。作者的硬體配置包括搭載 128GB 統一記憶體的 FEVM FAEX1（Strix Halo 平台）以及透過 OCuLink 連接的 RTX Pro 5000 Blackwell（48GB），可同時以 ROCm 跑 Qwen3.6 35b，並以 CUDA 跑 Qwen3.6 27b，主要用於 opencode 輔助開發。Warpdrv 的核心功能包含：管理多個 llama-server 實例、支援 MCP.json 與工具呼叫、內建 Model Router（可整合 claude-code 本地端）、以及實驗性的 KV-cache checkpointing。此工具不內建 llama.cpp 編譯檔，但提供一鍵 build 腳本（Recipes）。文中也詳細記錄了在 Ubuntu 25.10 上讓 Strix Halo iGPU 支援 ROCm 7.2 的完整步驟，包含核心升級至 6.18、GRUB 參數調整與 GTT 記憶體設定，對有意在 AMD APU 上跑本地 LLM 的使用者極具參考價值。目前為 Alpha 階段，開放 GitHub 回報問題與功能建議。
