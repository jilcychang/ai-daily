摘要

根據報導，OpenAI 與 Microsoft 在上個月重新談判合約後，雙方已同意將營收分潤金額設定上限為 380 億美元。此次重新協議是兩家公司長期合作關係中的重要里程碑。Microsoft 是 OpenAI 的主要投資方，多年來累計投入數百億美元，並獲得 OpenAI 技術的優先使用權與營收分成。此次設定上限，意味著 Microsoft 從 OpenAI 所能獲得的分潤將不超過 380 億美元，這對 OpenAI 未來轉型為營利公司的計畫具有關鍵意義。隨著 OpenAI 積極推動企業重組、吸引更多外部投資者，重新界定與 Microsoft 的財務關係有助於釐清股權結構，並為新投資人提供更清晰的獲利預期。此協議也反映出 OpenAI 在商業化道路上逐漸取得更大主導權。
