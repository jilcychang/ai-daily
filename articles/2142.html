GitHub Copilot 將於6月改採用量計費制，依 Token 消耗量收費

來源：iThome 新聞 · TW/zh-TW · 閱讀原文 ↗

摘要

GitHub 宣布旗下 AI 程式開發工具 Copilot 將於 2026 年 6 月 1 日起全面改為按用量計費模式。新制以 AI Credits 取代現行的進階請求（Premium Request）制度，計費基準改為 Token 消耗量，並依各模型公告的 API 費率扣抵 Credits。付費方案月費維持不變，但當用量超出方案內含的 AI Credits 後，須依各模型費率額外付費。此調整影響 Copilot 全系列方案，意味著重度使用者可能面臨更高費用，而輕度使用者則不受影響。此舉反映 GitHub 希望費用結構更貼近實際資源消耗，對企業用戶在成本預估與控管上將帶來新的挑戰。
