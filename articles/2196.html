DeepSeek 大幅降低 AI 模型價格，比 GPT-5.5 便宜最高達 97%

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

中國 AI 新創 DeepSeek 宣布大幅調降 API 使用費用，最新的 V4 系列模型在部分計費基準上，比 OpenAI 的 GPT-5.5 便宜最高達 97%，預示 AI 業界價格競爭將進一步白熱化。DeepSeek 於 4 月 27 日宣布，針對 API 用戶將「輸入快取命中（input cache hits）」費用降至原本的十分之一，最低輸入費用調降為每百萬 token 0.14 美元，且此調降為永久性措施。此外，為慶祝新旗艦模型 DeepSeek-V4-Pro 上市，DeepSeek 預計提供限時優惠至 5 月 5 日。此次降價被視為 DeepSeek 持續向 OpenAI、Google 等大廠施壓的策略，也再度引發外界對 AI 模型商業化定價模式的關注。對企業用戶而言，大幅壓低的 API 成本可能加速其採用 DeepSeek 模型的意願，並促使其他主要廠商跟進調整定價策略。
