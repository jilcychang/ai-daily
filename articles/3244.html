Anthropic 的 Claude Managed Agents 現在能「作夢」了

來源：Ars Technica — AI · US/en · 閱讀原文 ↗

摘要

Anthropic 在其 Code with Claude 開發者大會上，為 Claude Managed Agents 推出了名為「dreaming（作夢）」的新功能。這項功能目前處於研究預覽階段，僅限於 Claude Platform 上的 Managed Agents 使用。Managed Agents 是 Anthropic 提供的高階代理框架，建構於託管基礎架構之上，適合多個 agent 協作執行需耗時數分鐘至數小時的任務。所謂「作夢」，是指系統定期回顧近期的工作階段與記憶庫，篩選出值得長期保存的重要資訊，寫入「memory（記憶）」供未來任務參考。這項設計的出發點是解決 LLM 上下文視窗有限的問題——在長時間運作的專案中，關鍵資訊可能因超出上下文而遺失。這與對話模型常用的「compaction（壓縮）」機制類似，但 dreaming 更著重主動篩選與長期記憶的建立，而非單純縮減上下文長度。
