專為 LLM 設計的 Markdown 瀏覽器

來源：Reddit r/LocalLLaMA · US/en · 閱讀原文 ↗

摘要

一位開發者在 Reddit r/LocalLLaMA 分享了他打造的工具 TextWeb，這是一款專為 AI agent 設計的網頁渲染器。傳統做法是對網頁截圖後交給視覺模型處理，成本較高；TextWeb 改為將網頁內容渲染成 Markdown 格式，讓 LLM 能直接理解與操作，省去視覺模型的開銷。TextWeb 支援完整的 JavaScript 執行，並會標注互動元素，讓 LLM 能夠進行網頁導覽、上下捲動、在輸入欄位填寫文字、點擊按鈕等操作。工具提供 CLI 與 MCP 伺服器兩種使用方式，並與 llama.cpp 的網頁 UI 相容。此專案基於 Chris Robison 的 textweb（原版使用文字格狀渲染器）改造而來，原始碼已開源於 GitHub。對於想要打造低成本網頁自動化 agent、又不依賴視覺模型的開發者而言，這是一個值得關注的輕量替代方案。
