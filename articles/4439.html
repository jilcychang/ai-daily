Microsoft 開始取消 Claude Code 授權

來源：The Verge — AI · US/en · 閱讀原文 ↗

摘要

Microsoft 正準備大幅縮減內部的 Claude Code 授權，並將旗下開發者導向改用 Copilot CLI。據悉，Microsoft 自去年 12 月開始向內部數千名開發者開放 Claude Code（Anthropic 開發的 AI 程式碼工具）的使用權限，目的是讓專案經理、設計師等非工程師職能的員工也能嘗試寫程式。過去六個月來，Claude Code 在 Microsoft 內部廣受歡迎，但如今這波熱潮卻成為縮編的導火線。消息人士透露，Microsoft 即將撤銷大多數 Claude Code 授權，改推自家的 Copilot CLI。此舉反映出 Microsoft 在 AI 工具策略上的轉向——儘管與 Anthropic 維持合作關係，但在內部工具選擇上，仍優先扶植自家產品。這對 Anthropic 而言是一項不小的挫折，也顯示大型科技公司在採用第三方 AI 工具時，商業與競爭考量仍凌駕於實際效益之上。
