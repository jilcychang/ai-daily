在 Windows 上用原生 vLLM 讓 Qwen3.6-27B 跑出每秒 72 tokens（不需 WSL 或 Docker），附便攜啟動器與安裝程式

來源：Reddit r/LocalLLaMA · US/en · 閱讀原文 ↗

摘要

Reddit 用戶 /u/One_Slip1455 分享了一個開源專案，讓 Windows 用戶無需 WSL 或 Docker，即可原生運行 Qwen3.6-27B 模型。在 RTX 3090 上實測速度達短提示 72 tok/s、長提示（約 25k tokens）64.5 tok/s、單 GPU 127k 上下文 53.4 tok/s，雙 RTX 3090 可支援 160k 上下文。 安裝極為簡便：下載 ZIP 解壓後直接雙擊 `start.bat`，無需管理員權限、pip 或 Python 環境，首次執行會自動安裝依賴並可選擇從 HuggingFace 下載 Lorbus AutoRound INT4 量化模型，啟動後提供 OpenAI 相容的 API 端點。 作者為此專案建立了修補版 vLLM fork，解決了原生 Windows 上的若干相容性問題。支援 Ampere 與 Ada 架構（如 RTX 3090、4090、A6000），不支援 Pascal、Turing、Arc 或 AMD 顯示卡。RTX 50 系列（Blackwell）目前尚不支援，作者表示待 rebase 後會推出相應版本。對於想在 Windows 本地跑大模型但不想折騰虛擬化環境的用戶，這是目前最低門檻的解決方案之一。
