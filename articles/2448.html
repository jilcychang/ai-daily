Amazon 在 Bedrock 上推出 OpenAI 最新模型，並發布生產力產品套件

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

OpenAI 的模型終於登陸 Amazon Web Services（AWS）。此消息在 OpenAI 與 Microsoft（微軟）修改合作協議、允許其他雲端服務商銷售 OpenAI 模型後僅一天便宣布。AWS 執行長 Matt Garman 於舊金山活動上表示，GPT-5.4 目前已在 AWS 上提供有限預覽，最新模型 GPT-5.5 預計數週內上線，程式開發工具 Codex 也將陸續提供。Garman 強調，這是客戶長久以來的期待。此舉意味著 OpenAI 的模型生態系正式突破微軟獨家雲端的限制，進一步擴大到 AWS 的 Bedrock 平台，對企業用戶而言多了更靈活的選擇空間。
