將 Claude Code 打造成最強大的深度研究 Agent

來源：Reddit r/singularity · US/en · 閱讀原文 ↗

摘要

Reddit 用戶 heisdancingdancing 發布了開源專案 HyperResearch，這是一套建構在 Claude Code 上的技能框架，可將 Claude Code 轉化為深度研究代理（Deep Research Agent）。根據 DeepResearch Bench 評測，HyperResearch 在 agentic 搜尋領域的表現超越 OpenAI、Google 與 NVIDIA 的同類產品。只需一行指令即可安裝，並直接使用使用者現有的 Claude Code 訂閱，無需額外付費給 OpenAI 或 Gemini Pro。HyperResearch 採用 16 步驟的處理流程，每次研究會話都會建立可搜尋、可持久化的知識庫，並可在後續搜尋中累積擴展。框架內建事實查核、對抗性審查、廣度與深度調查等機制，確保研究結果盡可能貼近使用者的原始需求。底層使用 crawl4ai（開源 LLM 搜尋工具）擷取比標準網路搜尋更廣泛的資訊，並支援身份驗證會話設定，使 LinkedIn、Twitter 等需要登入的平台也能納入 agentic 搜尋範圍。適用場景包括股票交易策略研擬、競品分析、LLM 架構最新進展等大規模研究任務。
