Anthropic 與 SpaceX 達成合作，提升 Claude Code 使用上限

來源：Ars Technica — AI · US/en · 閱讀原文 ↗

摘要

Anthropic 在週三舉辦的 Code with Claude 開發者大會上宣布與 SpaceX 達成合作協議，將使用 SpaceX 位於田納西州曼菲斯資料中心的全部運算資源。執行長 Dario Amodei 表示，此合作旨在提升 Pro 和 Max 方案訂閱用戶的使用上限。隨著公告同步實施的調整包括：Claude Code 的五小時視窗限制加倍、取消尖峰時段的使用量限縮，以及提高 Opus 模型的 API 配額。
