Anthropic 將 Claude SDK 改為付費制以恢復 OpenClaw 存取，開發者批「使用額度縮減 25 倍」

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

Anthropic 宣布調整付費訂閱方案，推出新的「Agent SDK 積分」制度，允許 Claude 付費用戶再次透過外部 AI Agent 平台（包含 OpenClaw）使用 Claude 模型。此前 Anthropic 於今年 4 月起限制了第三方 Agent 平台的存取權限，OpenClaw 等工具因此受到波及。此次復開存取看似鬆綁，但開發者社群反應兩極——有人認為這是倒退，因為新制度附帶嚴格的使用上限，據稱相較過去縮減幅度高達 25 倍，實質上是「大幅縮水版的恢復」。開發者批評此舉雖名為開放，卻以積分配額變相限制使用量，對重度依賴 Claude 進行自動化工作流程的用戶衝擊明顯。
