Anthropic 收購 SDK 自動化新創 Stainless，OpenAI、Google 恐面臨服務終止

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

Anthropic 於 5 月 18 日宣布收購專注於 SDK 及 MCP 伺服器工具的開發者工具新創公司 Stainless，藉此強化 AI Agent 生態系的競爭力。Stainless 成立於 2022 年，總部位於紐約，主要提供 SDK 自動化生成服務。此次收購金額未公開，但據《The Information》報導，交易規模至少達 3 億美元（約新台幣 135 億元）。此收購最受矚目的影響在於：長期使用 Stainless 服務的 OpenAI 與 Google 等競爭對手，未來極可能無法繼續使用這項關鍵開發基礎設施，被迫自建替代方案。對 Anthropic 而言，將 Stainless 的技術整合至自身平台，不僅能加速 SDK 工具的開發效率，更能在 AI Agent 工具鏈上取得結構性優勢，進一步鞏固其在開發者生態系中的地位。
