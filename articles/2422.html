Anthropic 將 Claude Code 預估費用調漲一倍，預告轉向按量計費

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

Anthropic 近期大幅上調 Claude Code 的預估使用成本，實際上已預告將終止現行的月費訂閱制，轉向依使用量計費（隨用隨付）的收費模式。根據 Anthropic 於 4 月 27 日公布的數據，企業開發者使用 Claude Code 的平均費用約為每位每日活躍用戶 13 美元，相較幾週前公布的每日 6 美元暴漲超過一倍。換算成月費，每位開發者的使用成本約落在 150 至 250 美元之間。這意味著目前每月 20 美元的訂閱定價，早已遠低於實際產生的服務成本，現行方案顯然難以為繼。此次費用調整反映出 Claude Code 的實際使用量遠超預期，也讓外界開始關注 Anthropic 後續將如何重新定價，以及對開發者社群的實際衝擊。
