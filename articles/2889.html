「阻止失控的影子 AI」⋯MS 正式推出企業內部治理平台「Agent 365」

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

隨著 AI agent 在企業環境中快速擴散，Microsoft（MS）於 5 月 1 日正式推出專為管理 AI agent 設計的安全與治理平台「Agent 365」。MS 指出，AI agent 已從 Microsoft 365、Teams、Copilot 等既有工作工具，進一步蔓延至本地端 PC、雲端及 SaaS 等各類環境。隨著 agent 數量急速增加，企業面臨所謂「agent sprawl（代理擴散）」的問題，意即大量 agent 在企業的管控範圍之外自行運作，形成難以追蹤的「影子 AI」風險。Agent 365 的推出，正是為了因應這項挑戰，讓企業能夠統一監控、管理並治理在組織內部運行的各類 AI agent，防止失控情形發生。
