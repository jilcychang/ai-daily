Anthropic 阻止 Claude Code 外洩的困境之戰

來源：Heise Online · DE/de · 閱讀原文 ↗

摘要

Anthropic 內部一名員工因人為疏失，導致 Claude Code 的核心系統提示（system prompt）遭到外洩，進而揭露了其自主 AI 代理（autonomous AI agent）的底層架構設計。此次洩露引發兩大效應：其一，外界得以一窺 Anthropic 如何建構 Claude Code 的行為指引與工具使用框架；其二，事件迅速引爆關於 AI 生成程式碼著作權歸屬的激烈爭論。由於系統提示本身是否受版權保護、以及 AI 產出的程式碼由誰擁有，目前在法律上仍屬灰色地帶，此事件使相關討論更加白熱化。Anthropic 雖試圖透過法律手段要求下架外洩內容，但在網路擴散速度面前，此舉被外界視為勝算渺茫。此案不僅凸顯大型 AI 公司在智慧財產保護上的脆弱性，也預示著 AI 生成內容的版權規範，將成為業界與法律界未來必須正面迎戰的核心議題。
