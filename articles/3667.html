中國黑市以一折價格流通 Claude，成為「模型蒸餾溫床」

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

根據牛津大學中國政策研究所研究員 Zilan Chen 的揭露，中國存在一個稱為「中轉站」的 API 代理網路，讓中國開發者能以 Anthropic 官方定價約 10% 的低價取得 Claude 的存取權限。這些非法代理服務透過轉售 API 存取，繞過 Anthropic 對中國市場的使用限制。更令人憂慮的是，這類服務在提供低價存取的同時，可能蒐集大量使用者的輸入輸出資料，進而用於訓練或「蒸餾（distillation）」本土 AI 模型，實質上是以非授權方式汲取 Claude 的能力。此事件凸顯了先進 AI 模型在出口管制與智慧財產保護上的漏洞，也再次引發外界對中國如何利用西方 AI 技術加速自身模型發展的疑慮。
