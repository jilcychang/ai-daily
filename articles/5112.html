Anthropic 悄悄修補 Claude Code 2 個沙箱繞過漏洞，研究人員批評陷用戶於風險中

來源：iThome 新聞 · TW/zh-TW · 閱讀原文 ↗

摘要

Anthropic 旗下的 AI 程式開發工具 Claude Code，近期被研究人員揭露存在資安透明度問題。自 2025 年 10 月起的五個月內，Anthropic 悄悄修補了 Claude Code 兩個沙箱繞過（sandbox bypass）漏洞，卻未發布任何公告或安全通知，使廣大開發商與企業用戶在不知情的情況下，持續暴露於潛在攻擊風險中。研究人員對此提出強烈批評，認為 Anthropic 的做法不符合業界漏洞揭露的最佳實踐（responsible disclosure），應在完成修補後主動通知受影響用戶。沙箱繞過漏洞可能讓惡意攻擊者突破 Claude Code 的安全隔離機制，進而危害使用者系統。此事件凸顯 AI 工具在快速普及的同時，供應商的資安透明度與漏洞揭露政策同樣至關重要，企業在採用 AI 開發工具時，應更加重視供應商是否有完善的安全公告機制。
