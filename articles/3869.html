Show HN: adamsreview – 專為 Claude Code 打造的進階多代理 PR 審查工具

來源：Hacker News — Front Page (AI filter) · US/en · 閱讀原文 ↗

摘要

開發者 Adam Miller 發布了 adamsreview，一款 Claude Code 插件，透過平行子代理（sub-agents）、多階段驗證、持久化 JSON 狀態儲存，以及可選的 Codex CLI 整合，提供比內建工具更深入的 PR 程式碼審查。 作者表示，在自己的 PR 上測試，adamsreview 捕捉到的真實 bug 明顯多於 Claude 內建的 /review、/ultrareview、CodeRabbit、Greptile 與 Codex 內建審查，同時誤報率更低。 插件包含六個 slash 指令：review、codex-review、add、promote、walkthrough 與 fix。其中 walkthrough 指令利用 Claude 的 AskUserQuestion 功能，逐一引導使用者審視不確定的發現；fix 指令則會派發獨立代理針對每組問題進行修復，再用 Opus 模型重新審查，若有回退（regression）則自動還原，確保只提交通過驗證的修改。 與 /ultrareview 不同，adamsreview 運行在一般的 Claude Code 訂閱方案下（建議使用 Max plan），不會額外消耗 Extra Usage 額度，對重度使用者更具成本優勢。目前專案已開源，作者正在徵求 Claude Code 用戶與開發者的回饋。
