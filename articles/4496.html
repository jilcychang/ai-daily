GitHub Copilot 桌面應用程式開始技術預覽，將代理式開發整合到桌面工作環境

來源：iThome 新聞 · TW/zh-TW · 閱讀原文 ↗

摘要

GitHub 釋出 GitHub Copilot 桌面應用程式技術預覽版，讓開發者可從議題、Pull Request、提示詞或先前工作階段啟動 AI 代理開發任務，並在同一個環境中檢視程式碼變更、檢查結果與合併請求。此舉將 Copilot 的使用情境從程式碼撰寫輔助，擴展至可接續整個開發流程的桌面工作環境。根據官方預告，該應用程式預計於 6 月 2 日正式發布。
