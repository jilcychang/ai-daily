Launch HN：Voker（YC S24）——AI Agent 的分析平台

來源：Hacker News — Front Page (AI filter) · US/en · 閱讀原文 ↗

摘要

Voker.ai 是由 Alex 與 Tyler 共同創辦、入選 YC S24 的 AI agent 分析平台，專為 AI 產品團隊提供 agent 效能可視化工具。 現有問題是：AI 產品團隊在 production 環境中幾乎無法主動掌握 agent 的表現，只能靠用戶抱怨才知道出問題，再盲目修改 prompt、祈禱不會壞掉其他地方，週而復始。Voker 調查逾 90% 的 YC 創辦人都有相同困境。 市場上現有的 observability（可觀測性）工具只適合工程師做個別 trace 除錯；evals（評估）工具只能測試已知問題；傳統產品分析工具又沒針對對話型 agent 設計。把 log 丟給 Claude 或 ChatGPT 請它摘要，也因 LLM 不擅長統計而不可靠。 Voker 提出三個 agent 分析原語：Intents（用戶意圖）、Corrections（用戶修正）、Resolutions（意圖是否被解決）。平台透過 LLM 與階層式文字分類，自動對每段對話標注並聚合成動態類別，讓團隊不必逐一閱讀對話就能掌握用戶使用趨勢與 agent 失敗模式。核心資料工程刻意不依賴 LLM，確保統計結果一致且可重現。 Voker 提供輕量 SDK，支援 OpenAI、Anthropic、Gemini，Python 與 TypeScript 皆可用。免費方案每月 2,000 events，付費方案從每月 $80 起，含 30 天免費試用。
