inclusionAI 開源兆參數旗艦模型 Ling-2.6-1T

來源：Reddit r/LocalLLaMA · US/en · 閱讀原文 ↗

摘要

inclusionAI 正式開源旗艦模型 Ling-2.6-1T，這是一個擁有兆（1 trillion）參數規模的大型語言模型，專為複雜現實場景設計，特別強化了程式開發與日常工作流程的應用能力。 主要升級亮點包含四項：一、採用 MLA 與 Linear Attention 混合架構，大幅降低長上下文推理的延遲與 VRAM 使用量，提升吞吐量並降低每 token 運算成本；二、引入「快速思考」（Fast Thinking）機制，透過訓練後階段的 Contextual Process Redundancy Suppression 獎勵策略，減少冗長的思維鏈（CoT），直接壓縮輸出成本；三、在多個高難度基準測試上達到開源 SOTA，包含 AIME26、SWE-bench Verified、BFCL-V4、TAU2-Bench 及 IFBench；四、可與 Claude Code、OpenClaw、OpenCode、CodeBuddy 等主流 agent 框架無縫整合，適合企業級多工具、多步驟的自動化流程。 Ling-2.6-1T 目前已在 Hugging Face 上公開，對本地端部署超大模型有興趣的開發者值得關注。
