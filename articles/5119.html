Confluent 推出 MCP 與 Agent Skills，讓 AI 代理協助管理資料串流作業

來源：iThome 新聞 · TW/zh-TW · 閱讀原文 ↗

摘要

IBM 旗下資料串流業者 Confluent 宣布為其 Confluent Intelligence 與 Confluent Cloud 平台新增多項功能，包括全託管 MCP（Model Context Protocol）伺服器、Confluent Agent Skills、個資偵測與遮蔽，以及 Azure Private Link 私有連線支援。透過 MCP 伺服器與 Agent Skills，企業可讓 AI 代理自動化管理 Kafka 資料串流作業，降低人工維運負擔；個資偵測與遮蔽功能則可即時識別並保護串流中的敏感資料，防止外洩；Azure Private Link 則讓企業繞開公開網路，透過私有連線存取 Confluent Cloud，強化資安防護。這些功能主要針對正在建置即時 AI 應用的企業，協助其在提升開發效率的同時，兼顧資料治理與合規需求。
