OpenAI 為「Codex」推出 Chrome 擴充功能，可直接控制瀏覽器

來源：AI Times (Korea) · KR/ko · 閱讀原文 ↗

摘要

OpenAI 於 5 月 8 日發布適用於 macOS 與 Windows 的「Codex Chrome Extension」，讓 AI 編程助手 Codex 能在使用者的 Chrome 瀏覽器環境中直接運作。此擴充功能主要針對過去僅靠 API 或外掛程式難以處理的瀏覽器中心任務而開發。OpenAI 指出，自先前推出「Computer Use」功能後，越來越多使用者傾向在真實瀏覽器環境中操作 AI 代理人，相關需求因此大幅增加。此次更新讓 Codex 從桌面端延伸至瀏覽器端，進一步擴大其作為 AI 代理人的應用場景。
