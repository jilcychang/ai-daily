GitHub 從 Copilot 方案中移除免費模型

來源：Heise Online · DE/de · 閱讀原文 ↗

摘要

GitHub 調整了 Copilot 的收費方式，從原本的固定 Premium 請求次數計費，改為依實際 token 使用量計費。此次改版同時取消了免費模型的提供。這項變更意味著使用者在選擇不同 AI 模型時，將直接根據消耗的 token 數量付費，而非享有固定的免費額度。對於大量使用 Copilot 的開發者來說，新的計費模式可能帶來成本結構上的明顯變化。
